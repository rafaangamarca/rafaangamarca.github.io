
# FIT PUSH 🏋️‍♂️ - Plataforma de Entrenamiento y Nutrición
Fit Push es una aplicación web integral diseñada para entusiastas del fitness que buscan optimizar su rendimiento mediante herramientas basadas en ciencia y una gestión técnica del entrenamiento.

🚀 Descripción del Proyecto
Este proyecto nace con el objetivo de centralizar las herramientas esenciales para cualquier deportista en una interfaz moderna, oscura y de alto contraste (Verde Neón). La plataforma permite desde la consulta técnica de ejercicios hasta la planificación automatizada de rutinas y el cálculo de requerimientos nutricionales.

🛠️ Funcionalidades Principales
Biblioteca de Movimientos: Base de datos de ejercicios con guías técnicas, clasificación por grupos musculares (Pectoral, Espalda, Pierna, etc.) y niveles de dificultad.

Generador de Rutinas Inteligente: Sistema que diseña planes semanales personalizados (Full Body, Torso/Pierna o PPL) basados en la disponibilidad de días y tiempo del usuario.

Calculadora de Macros: Herramienta que utiliza la fórmula de Mifflin-St Jeor para determinar las calorías de mantenimiento y el reparto de macronutrientes (proteínas, grasas y carbohidratos).

Guía de Suplementación: Sección informativa sobre suplementos con mayor evidencia científica (Creatina, Whey Protein, Pre-entreno) clasificados por nivel de eficacia.

Modo Entrenamiento: Interfaz interactiva para seguir la rutina en tiempo real, incluyendo un cronómetro de descanso integrado.

💻 Tecnologías Utilizadas
Frontend: HTML5, CSS3 (Custom Variables, Animaciones).

Framework de Estilos: Bootstrap 5.3.

Interactividad: JavaScript Vanilla (Manipulación del DOM, lógica de cálculo nutricional y gestión de estados de rutina).

Iconografía y Fuentes: FontAwesome y Google Fonts (Oswald & Roboto).

SEO & Analytics: Configuración de meta-tags para Open Graph, Twitter Cards, Sitemap XML y Google Analytics.
